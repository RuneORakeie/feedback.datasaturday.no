Feedback utility for conference sessions
What is this? This is a utility that helps conference organizers and speakers collect session feedback. I'm an organizer. How do I get started? You can import all your sessions from the Sessionize.com API endpoint in a few simple steps. Once the import is complete, the import utility will give you a secret key that you can use to log on to the admin interface, where you can copy or download QR codes and direct links to the evaluation forms. Can I apply my own graphics and stylesheet? Yes. Before you create and import your event, create a CSS file (and, optionally, a logo image) and put them in a pull request to the "test" branch of the project. I have questions/suggestions/feedback? Please contribute to the Github repository or create an issue.

Powered by an open-source session evaluation app.

This site doesn't track you, and it doesn't save personally identifying information, unless you type it yourself in the evaluation fields.
